    ⌨️ 按键映射 #

    LazyVim 使用 which-key.nvim 帮助你记住你的按键映射。 按任意键，如 <space>，你将看到一个包含所有以 <space> 开始的按键映射的弹窗。

    • 默认的 <leader> 键是 <space>
    • 默认的 <localleader> 键是 \
    模式描述
    nNormal Mode
    vVisual Mode
    sSelect Mode
    iInsert Mode
    cCommand Mode
    xEx Mode
    tTerminal Mode
